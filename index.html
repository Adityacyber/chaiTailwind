chai.js

A lightweight utility-first CSS engine built in pure JavaScript. Write chai-* classes, get inline styles — no build step, no config, no runtime overhead.

View Utility → View Reference

QUICK START

<!-- 1. Include the script --> <script src="chai.js"></script> <!-- 2. Use chai-* classes on any element --> <div class="chai-flex chai-gap-4 chai-p-6 chai-bg-indigo chai-rounded-xl"> <h2 class="chai-text-white chai-font-bold chai-text-2xl">Hello chai!</h2> </div> <!-- 3. chai.js scans the DOM on load, applies inline styles, and removes the original class names. Done! -->

How it works

In four clean steps

STEP 01
DOM Ready
chai.js listens for DOMContentLoaded (or runs immediately if the DOM is already ready).
STEP 02
Scan Classes
Queries every element with a [class*="chai-"] attribute and loops their classList.
STEP 03
Parse & Match
Each chai-* token is stripped of the prefix and matched against 120+ regex rules.
STEP 04
Apply Styles
Resolved CSS properties are batch-applied as el.style.* inline styles. Class names are removed.
COMMING SOON...
Mutation Observer
A MutationObserver watches for dynamically added nodes and class changes — real-time support out of the box.

Utility Reference

A complete index of supported utilities. Scale = 4px (e.g. chai-p-4 → 16px).

Category Class Pattern CSS Output Example

Padding X chai-px-{n} padding: 0 {n×4}px chai-px-6 → 0 24px

Padding Top chai-pt-{n} padding-top: {n×4}px chai-pt-2
Margin chai-m-{n} margin: {n×4}px chai-m-4 → 16px
